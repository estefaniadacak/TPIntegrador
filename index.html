Análisis y Modelos Predictivos — Rigidez Lateral

Objetivo

Evaluar si el período natural estimado permite obtener predicciones de rigidez lateral comparables a las obtenidas con el período medido, utilizando modelos explicativos y predictivos aplicados a galpones industriales.

Ir a Modelos

Introducción

La rigidez lateral es un parámetro clave para la respuesta estructural ante viento y sismo. El período natural medido brinda alta precisión, pero no siempre puede obtenerse. Este trabajo analiza si el período estimado puede reemplazarlo para prediseño.

Conclusiones

  • El período natural es la variable más influyente.
  • Random Forest y XGBoost tuvieron mejor performance.
  • El período estimado es un buen sustituto para prediseño.
